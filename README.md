# Mastermind

The game Mastermind

## Installation

Add this line to your application's Gemfile:

```
$ git clone https://github.com/joemsak/mastermind
$ cd mastermind
$ bundle
$ rake install
```
## Run the game after installation above

```
$ mastermind
```
